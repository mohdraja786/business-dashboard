# Business Dashboard

यह एक वेब-आधारित बिजनेस डैशबोर्ड है जो सेल्स रिपोर्ट का विश्लेषण करने के लिए बनाया गया है। यह प्रोजेक्ट Raja Enterprises के डेटा को विजुअलाइज करने में मदद करता है।

## विशेषताएं (Features)
- सेल्स डेटा का ग्राफिकल रिप्रेजेंटेशन।
- CSV फाइलों के जरिए आसान डेटा अपलोड।
- उपयोग में आसान इंटरफेस।

## कैसे शुरू करें (How to Run)
1. रिपॉजिटरी को क्लोन करें।
2. अपनी मशीन पर Flask इंस्टॉल करें:
   `pip install flask pandas`
3. एप्लीकेशन चलाएं:
   `python app.py`

## तकनीकी जानकारी (Tech Stack)
- **Framework:** Flask (Python)
- **Data Handling:** Pandas
- **Frontend:** HTML/CSS
